# 클론코딩🌱velog
  개발자들을 위한 블로그 서비스 벨로그를 클론코딩
  
  ## 작업기간
  <strong>2022.02.18 ~ 2022.02.24</strong>   
     
  ## 팀원
  <strong>FE</strong>: 이규리(조장)  송원석   
  <strong>BE</strong>: 김승호  장현광  정한나   
     
  ## 구현기능
  ### 회원관리
  - 로그인, 회원가입 모달창 구현   
  - 로그아웃

  ### 메인페이지
  - 로그인 여부에 따라 변하는 헤더
  - 서버에서 불러와 메인페이지에 뿌려주기  
  - 클릭한 게시물의 상세페이지 이동
   
  ### 상세페이지
  ☑️ 게시글 상세   
  - 게시물 수정 및 삭제
  - 좋아요 버튼 (진행중)
     
  ☑️ 댓글   
  - 댓글 작성 및 수정, 삭제 (진행중)
     
  ### 글작성 페이지
  - 마크다운 에디터 적용
